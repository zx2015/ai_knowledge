# 良性与恶性肿瘤鉴别指南

肿瘤是机体细胞在致瘤因素作用下，基因突变导致细胞异常增生而形成的新生物。临床上将其分为良性（Benign）和恶性（Malignant）两大类。

## 1. 核心区别汇总

| 鉴别点 | 良性肿瘤 | 恶性肿瘤 (癌/肉瘤) |
| :--- | :--- | :--- |
| **组织分化** | 分化成熟，异型性小，与原组织相似 | 分化不成熟，异型性大，与原组织差异大 |
| **核分裂象** | 无或极少，不出现病理性核分裂 | 多见，常出现病理性核分裂 |
| **生长速度** | 缓慢 | 迅速 |
| **生长方式** | 膨胀性或外生性生长，常有包膜 | 浸润性或外生性生长，无包膜 |
| **转移性** | **不转移** | **常有转移** (淋巴、血液、种植) |
| **复发性** | 手术切除后很少复发 | 手术切除后易复发 |
| **对机体影响** | 较小，主要为局部压迫或阻塞 | 严重，破坏组织、出血、感染、恶病质 |

## 2. 浸润与转移机制

恶性肿瘤之所以危险，在于其具有**浸润性**和**转移性**：
1. **浸润性**：恶性肿瘤细胞分泌蛋白酶，降解周围组织的基底膜，像树根一样长入邻近器官。
2. **转移路径**：
    - **淋巴道转移**：多数癌（上皮组织恶性肿瘤）的首发途径。
    - **血行转移**：多见于肉瘤，常通过静脉进入肺、肝、骨等。
    - **种植性转移**：体腔内器官肿瘤细胞脱落，种植于其他器官表面（如胃癌种植到卵巢）。

## 3. 临床预警信号 (C.A.U.T.I.O.N)

虽然良恶性需病理确诊，但以下迹象需警惕恶性可能：
- **C**hange in bowel or bladder habits (排便、排尿习惯改变)
- **A** sore that does not heal (经久不愈的溃疡)
- **U**nusual bleeding or discharge (异常出血或分泌物)
- **T**hickening or lump (身体部位出现硬块/肿块)
- **I**ndigestion or difficulty swallowing (消化不良或吞咽困难)
- **O**bvious change in a wart or mole (痣或疣出现明显变化)
- **N**agging cough or hoarseness (持续咳嗽或嘶哑)

## 4. 诊断“金标准”

- **病理检查**：通过穿刺或切除获取组织，进行显微镜下观察及免疫组化分析。
- **辅助检查**：
    - **影像学**：CT、MRI 观察边界，PET-CT 观察代谢活性。
    - **肿瘤标志物**：如 AFP（肝癌）、CEA（结直肠癌）等，用于辅助参考和随访。
