# 量化因子与因子库详解

因子（Factor）是量化交易中最小的逻辑单元。因子库则是对这些逻辑进行系统化管理、回测和组合的平台。

## 1. 因子的定义
因子是一个将原始数据映射为预测值的函数：
$$F = f(Data_{raw})$$
一个有效的因子必须在统计上能够解释资产收益率的差异。

## 2. 因子库的四大支柱

### 2.1 动量因子 (Momentum)
- **逻辑**：强者恒强。
- **代表**：过去 N 日收益率、股价创新高距离。

### 2.2 价值因子 (Value)
- **逻辑**：均值回归，买入被低估的资产。
- **代表**：PE (市盈率), PB (市净率), EV/EBITDA。

### 2.3 质量因子 (Quality)
- **逻辑**：买入赚钱能力强、财务健康的公司。
- **代表**：ROE (净资产收益率), 资产负债率。

### 2.4 情绪因子 (Sentiment) - **LLM 核心领域**
- **逻辑**：市场参与者的心理会驱动价格。
- **来源**：新闻情绪得分、研报预期调整、换手率异常。

#### 情绪因子生成流程：
1. **数据清洗**：从 API 获取原始文本，去除广告和重复新闻。
2. **LLM 映射**：使用 `Structured Output`（如 JSON）让 LLM 对每条新闻给出 [-1, 1] 之间的连续分值。
3. **加权聚合**：根据新闻源权威度或新闻热度，将当日多条分值合成为单一股票的日度分值。
4. **截面标准化**：通过 Z-Score 或分位数转换，确保因子在全市场股票间具有可比性。

## 3. 因子评价体系 (如何判断因子好坏？)
- **IC (Information Coefficient)**：因子值与未来收益率的相关系数。> 0.05 通常被认为有效。
- **ICIR**：IC 的均值除以标准差，衡量因子的稳定性。
- **换手率 (Turnover)**：因子信号改变的频率，决定了交易成本。

## 4. 工业级工具
- **Qlib Alpha158**：微软提供的 158 个基础 AI 因子集合。
- **Alphalens**：Python 下经典的因子分析工具。
- **WorldQuant Alpha101**：101 个手工定义的非线性因子公式。

## 5. LLM 在因子库中的进化
- **自动化挖掘**：利用 LLM 自动阅读金融文献，提取其中的数学公式并转化为因子代码。
- **语义因子化**：将非结构化文本（如 CEO 演讲）通过 LLM 向量化，提取传统模型无法识别的特征。
