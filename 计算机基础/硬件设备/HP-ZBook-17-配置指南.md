# HP ZBook 17 系列移动工作站配置指南

HP ZBook 17（后期更名为 ZBook Fury 17）是 HP 旗下的旗舰级移动工作站，定位于“移动桌面替换者”（Desktop Replacement）。

## 1. 型号演变
- **ZBook 17 G1 - G6**：经典的 17.3 英寸 16:9 比例工作站。
- **ZBook Fury 17 G7 - G8**：更紧凑的窄边框设计。
- **ZBook Fury 16 (G9+)**：17 英寸系列的继任者，改用 16 英寸 16:10 屏幕。

## 2. 经典型号配置对比

| 型号 | ZBook 17 G3 (里程碑代) | ZBook Fury 17 G8 (最后巅峰代) |
| :--- | :--- | :--- |
| **发布年份** | 2016 年 | 2021 年 |
| **处理器 (CPU)** | 最高 Intel Xeon E3-1575M v5 或 i7-6820HQ | 最高 Intel Xeon W-11955M 或 i9-11950H |
| **显卡 (GPU)** | 最高 NVIDIA Quadro M5000M (8GB) | 最高 NVIDIA RTX A5000 (16GB) |
| **内存 (RAM)** | 4 插槽, 最高 64GB DDR4-2133 | 4 插槽, 最高 128GB DDR4-3200 |
| **硬盘 (SSD)** | 2 个 M.2 NVMe 插槽 | 4 个 M.2 NVMe 插槽 |
| **接口** | 首次引入 **雷电 3 (Thunderbolt 3)** | 升级为 **雷电 4 (Thunderbolt 4)** |
| **设计特色** | 机身大幅减重, 取消光驱 | 窄边框、高性能 Vapor Chamber 散热 |

## 3. 设计理念
- **免工具拆卸**：大部分 ZBook 17 型号支持底盖免工具开启，方便更换内存和硬盘。
- **ISV 认证**：确保在 AutoCAD、Maya、SolidWorks 等专业软件上的稳定性。
- **MIL-STD 测试**：通过多项美国军标测试，具备极高的耐用性。
